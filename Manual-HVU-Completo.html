Introducción

Descripción del Software

La Aplicación Gestión Dormitorios HVU es una herramienta móvil desarrollada para optimizar la administración y el seguimiento de actividades dentro del Hogar de Varones Universitarios (HVU) de la Universidad Linda Vista. El sistema permite gestionar aspectos clave de la vida residencial, como el registro de asistencias, la evaluación de limpieza, y el manejo de incidencias disciplinarias.

Objetivo del Manual

El propósito de este documento es proporcionar a los usuarios finales una guía clara y detallada sobre la instalación y el uso de las funcionalidades principales de la aplicación, independientemente de su rol asignado dentro del sistema.

Público Objetivo

  • Estudiantes: Residentes del Hogar de Varones Universitarios.
  • Monitores: Estudiantes designados con responsabilidades de supervisión específicas.
  • Preceptores: Personal administrativo responsable de la gestión integral del dormitorio.

Guía de Instalación

Requisitos Previos

  • Dispositivo Móvil: Un teléfono inteligente o tableta compatible con Android 6.0+ o iOS 12.0+.
  • Conectividad: Acceso a Internet mediante Wi-Fi o datos móviles.
  • Hardware:
    • Cámara funcional para escaneo QR.
    • Mínimo 4 GB de RAM.

Procedimiento de Instalación

  1. Acceda a Google Play Store o App Store.
  2. Busque “Gestión Dormitorios HVU-ULV”.
  3. Seleccione la aplicación correcta.
  4. Pulse “Instalar”.
  5. Acepte los permisos solicitados.
  6. Espere la instalación.
  7. Abra la aplicación desde el menú de su dispositivo.

Verificación de Instalación

La instalación se considera exitosa si la aplicación se abre correctamente y muestra la pantalla de autenticación.

Uso de las Funcionalidades Principales

La aplicación mostrará funcionalidades diferentes según el rol del usuario autenticado.

Funciones Comunes

Autenticación (Inicio de Sesión)

  1. Abra la aplicación.
  2. Regístrese si es la primera vez.
  3. Ingrese su Identificador de Usuario.
  4. Ingrese su Contraseña.
  5. Pulse “ACCEDER”.

Consulta y Edición de Perfil

  1. Abra el menú lateral.
  2. Seleccione “Perfil”.

Configuración de Apariencia

  1. Abra el menú lateral.
  2. Seleccione “Configuración”.
  3. Active o desactive “Modo Oscuro”.

Cierre de Sesión

  1. Abra el menú lateral.
  2. Seleccione “Cerrar sesión”.

Funcionalidades para Estudiantes

Consulta de Evaluación de Limpieza

  1. Pulse “Limpieza”.
  2. Visualice la fecha, monitor, criterios y calificación.

Registro de Asistencia mediante QR

  1. Pulse “Asistencia”.
  2. Seleccione “Escanear código QR”.
  3. Permita el acceso a la cámara.
  4. Escanee el QR.

Consulta de Amonestaciones

  1. Pulse “Amonestación”.
  2. Consulte su historial.

Consulta de Reportes Disciplinarios

  1. Pulse “Reportes”.
  2. Consulte la lista de reportes.

Funcionalidades para Monitores

Generación de Código QR para Asistencia

  1. Pulse “Generar QR”.
  2. Seleccione el tipo de actividad.
  3. Muestre el código generado.

Consulta de Asistencias Registradas

  1. Pulse “Ver Asistencias”.
  2. Seleccione tipo de culto y fecha.

Registro de Evaluación de Limpieza

  1. Pulse “Limpieza”.
  2. Seleccione un cuarto y califique criterios.

Gestión de Reportes

  1. Pulse “Reportes”.
  2. Busque por matrícula.
  3. Pulse “+” para crear un reporte.

Funcionalidades para Preceptores

Gestión de Roles de Monitor

  1. Pulse “Monitores”.
  2. Asigne o elimine roles.

Consulta General de Reportes

  1. Pulse “Reportes”.
  2. Busque o desplácese para ver más información.

Creación de Reporte Disciplinario

  1. Pulse “+”.
  2. Ingrese matrícula y motivo.
  3. Pulse “Guardar”.

Supervisión del Estado de Limpieza General

  1. Pulse “Limpieza”.
  2. Revise calificaciones y estados.

Gestión de Amonestaciones

  1. Pulse “Amonestaciones”.
  2. Registre o consulte amonestaciones.

Preguntas Frecuentes (FAQ)

  • ¿Cómo cambio mi contraseña? Contacte al Preceptor.
  • ¿Qué significan los niveles de amonestación? Leve, Media y Grave.
  • ¿Puedo editar calificaciones? Solo el Preceptor puede hacerlo.

Solución de Problemas Comunes

  • Error al cargar datos: Revise su conexión a Internet.
  • Matrícula no existe: Verifique el número ingresado.
  • Datos desactualizados: Deslice hacia abajo para refrescar.
  • La app se cierra: Reinicie o actualice la aplicación.
